# Data description

## Synthetic data

Synthetic instances are generated by `generate_synthetic_shp_data.m`. Each item
has a three-point marginal support. The marginal probabilities are determined
from the moment construction used in the paper, and the chain-supported
worst-case distribution is built by `build_chain_worst_distribution.m`.

## PJM data

The real-data experiment uses PJM Data Miner 2:

- Day-Ahead Hourly LMPs
- Real-Time Hourly LMPs

Date range:

- May 1, 2025 to August 1, 2025

Test period:

- July 2025

For each test day, the previous 60 days of realized real-time LMP vectors are
used to build the empirical training distribution. The day-ahead LMPs define
the first-stage prices, and the realized real-time LMPs define the second-stage
values.

The experiment uses 12 PJM hubs:

- AEP GEN HUB
- AEP-DAYTON HUB
- ATSI GEN HUB
- CHICAGO GEN HUB
- CHICAGO HUB
- DOMINION HUB
- EASTERN HUB
- N ILLINOIS HUB
- NEW JERSEY HUB
- OHIO HUB
- WEST INT HUB
- WESTERN HUB

Peak hours:

- 8--19

Negative LMPs are retained without truncation.
