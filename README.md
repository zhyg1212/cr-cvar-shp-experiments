# Numerical experiments for CR-CVaR-SHP

This repository contains the MATLAB code and data used for the numerical
experiments in the paper on correlation-robust and CVaR-based Sell-or-Hold
Problems under marginal ambiguity.

## Requirements

- MATLAB R2020b or later.
- No commercial optimization solver is required for the reported experiments.

The experiments reported in the manuscript were run in MATLAB R2020b on a
Windows 11 64-bit machine with a 13th Gen Intel(R) Core(TM) i5-13500H processor
and 16 GB RAM.

## Repository contents

- `generate_synthetic_shp_data.m`: synthetic three-point moment ambiguity data
  generator.
- `build_chain_worst_distribution.m`: construction of the chain-supported
  worst-case coupling.
- `tail_truncate_distribution.m`: construction of the lower-tail truncated
  distribution for the CVaR reformulation.
- `solve_shp_approximation.m`: candidate-selection approximation algorithm.
- `solve_shp_forward_greedy.m`: forward/backward policy construction used in
  the policy-comparison experiment.
- `solve_shp_enumeration.m`: exact enumeration solver for small SHP instances.
- `solve_shp_cvar_direct_enumeration.m`: direct exact CVaR enumeration solver.
- `evaluate_shp_policy.m` and `evaluate_shp_policy_dist.m`: out-of-sample
  policy evaluation routines.
- `run_*.m`: scripts used to generate the numerical tables.
- `da_hrl_lmps.csv` and `rt_hrl_lmps.csv`: PJM day-ahead and real-time hourly
  LMP data used in the real-data experiment.
- `results/`: CSV summaries and raw result files generated by the scripts.
- `figures/`: generated figures.

## Main experiments

Run the following scripts from this directory in MATLAB.

```matlab
run_cvar_reformulation_experiment
run_scalability_experiment
run_approximation_ratio_experiment
run_policy_comparison_forward_experiment
run_pjm_realdata_experiment
```

The scripts write their outputs to the `results/` folder.

## Synthetic experiments

The synthetic instances use three-point marginal distributions. The nominal
distribution is generated from nominal marginals, while the correlation-robust
models use the marginal ambiguity set induced by the three-point moment
construction. The chain-supported distribution `Pstar` is constructed once from
the empirical or synthetic marginals and then used by the CR-SHP reformulation.
For CVaR-CR-SHP, the lower-tail truncated distribution is built from `Pstar`.

The main synthetic policy-comparison table is generated by:

```matlab
run_policy_comparison_forward_experiment
```

This script uses ten random seeds, `n = 60`, `k = 30`, `alpha = 0.75`,
`lambda = 0.75`, 400 nominal training scenarios, and 2000 out-of-sample test
scenarios for the sampled dependence structures.

## PJM real-data experiment

The PJM experiment uses Day-Ahead Hourly LMPs and Real-Time Hourly LMPs from
PJM Data Miner 2 for May 1 to August 1, 2025. The test period is July 2025.

The script uses 12 PJM hubs and peak hours 8--19, so each test day contains
144 hub-hour items and the budget is set to 72. For each test day, the nominal
distribution is the empirical joint distribution of the previous 60 days of
realized real-time LMP vectors. The marginal ambiguity set is constructed from
the empirical marginal distribution of each hub-hour item.

Run:

```matlab
run_pjm_realdata_experiment
```

The script expects the PJM CSV files to be available as:

```text
da_hrl_lmps.csv
rt_hrl_lmps.csv
```

or under `data_pjm/` as described in `pjm_realdata_readme.md`.

## Notes on data

The PJM data files in this repository are exported from PJM Data Miner 2. If
you prefer to reproduce the data download manually, use the Day-Ahead Hourly
LMPs and Real-Time Hourly LMPs feeds for the same date range and hub filters.

Negative LMPs are retained without truncation and enter the model directly as
negative first-stage or second-stage profits.
